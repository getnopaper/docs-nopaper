## How Rewards Work

### Reward Pool

NoPaper receives B3TR allocations from VeBetterDAO's reward pool each **cycle** (~1 week). These tokens are distributed to verified submissions.

### Base Reward Calculation

Your base reward depends on:

1. **Wallet Trust Level** (based on wallet age)
2. **Receipt Quality** (items purchased)
3. **Environmental Impact** (sustainability score)

```
Final Reward = Base Reward √ó Ecosystem Multiplier
```

### Reward Ranges by Trust Level

| Trust Level | Min Reward | Max Reward |
|-------------|------------|------------|
| Newcomer (&lt;30 days) | 0.01 B3TR | 0.05 B3TR |
| Established (30-90 days) | 0.05 B3TR | 0.5 B3TR |
| Trusted (90-180 days) | 0.2 B3TR | 1.0 B3TR |
| Veteran (&gt;180 days) | 0.5 B3TR | 2.0 B3TR |

## Reward Factors

### Receipt Value
- Small purchases (&lt;$20): Closer to minimum
- Medium purchases ($20-50): Middle range
- Large purchases (&gt;$50): Closer to maximum

### Eco-Friendly Bonus
- Organic stores: +20% bonus
- Local farmers markets: +20% bonus
- Certified sustainable products: +20% bonus
